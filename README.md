# Sistem-Ontologi-Parfum-untuk-Pencarian-Aroma
Proyek ini membangun sistem ontologi parfum menggunakan OWL untuk merepresentasikan komposisi aroma seperti top, middle, dan base notes. Tujuannya adalah mengubah data parfum menjadi pengetahuan terstruktur yang dapat digunakan untuk eksplorasi dan pencarian yang lebih cerdas.
